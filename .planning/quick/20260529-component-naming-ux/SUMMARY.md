# Component naming + segmentation UX

Renamed all 23 block types from journalism jargon (The Lede, Witness, Chronicle,
Epilogue, Immersive…) to plain language anyone understands (Cover, Quote, Timeline,
Ending, Full-Screen Banner…). Evocative descriptions kept for flavor.

Regrouped the Add-block palette from 6 mood-based categories into 6 intent-based ones:
Page Structure · Text · Images & Media · Scroll Animations · Data & Facts · Embeds.
