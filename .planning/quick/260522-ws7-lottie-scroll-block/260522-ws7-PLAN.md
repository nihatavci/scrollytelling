# LottieScroll Block

## Goal
Add `LottieScroll` as a new block type that plays a Lottie JSON animation scrubbed by scroll position (scroll-driven), or optionally on autoplay when in view. No npm deps — loads lottie-web lazily from CDN inside render.js.

## Files
- Modify: `js/render.js` — add `loadLottieWeb()` lazy CDN loader + `renderLottieScroll()` + BLOCK_RENDERERS entry
- Modify: `admin/ui/app.js` — BLOCK_SCHEMAS, BLOCK_ICONS, PALETTE_CATEGORIES, BLOCK_PREVIEWS, creation card defaults
- Modify: `admin/ui/styles.css` — LottieScroll block CSS
- Modify: `js/visual-edit.js` — EDITABLE_MAP caption entry

## CDN
`https://cdn.jsdelivr.net/npm/lottie-web@5.12.2/build/player/lottie_light.min.js`
(SVG renderer only, ~115KB gzipped, sufficient for all Lottie use cases)
Lazy-loaded by render.js on first LottieScroll render — no admin/index.html changes needed.

## Block Data Shape
```json
{
  "lottieUrl": "",
  "caption": "",
  "layout": "contained",
  "scrubMode": "scroll"
}
```
